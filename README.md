Darulhikma Website

Overview

This is the official website for **Darulhikma Organization**, an Islamic and academic initiative based in Dodola. It presents the organization’s history, achievements, activities, and contact information.

Features

- Responsive design using Bootstrap
- Hero section with background image
- Expandable History & Achievements
- Gallery with category-based modal
- Contact form for user messages

Technologies

- HTML
- CSS
- JavaScript
- Bootstrap

Setup

1. Download or clone the project
2. Open `index.html` in your browser
3. Add your images in the `images` folder

Mission

To spread Islamic and academic knowledge through education, training, and community service.
